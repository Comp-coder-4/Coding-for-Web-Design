HTML

HyperText Markup Language. This is the backbone, or skeleton, of web pages and is absolutely essential for any web developer. Although it's not a programming language, it is responsible for holding the structure of a web page.

Topics include:

  • elements
  • attributes
  • headings
  • paragraphs
  • links
  • tables
  • lists
  • classes
  • div
  • id

HTML on its own can bring about loads of content, but on its own can bring about quite a dull-looking page...

So what's the solution?


CSS

Cascading Style Sheets. Get the power to add some style to your web page. With CSS, you can bring your web pages to life with colours, crisp layouts and, overall, an aesthetically pleasing look!


Topics include

  • selectors
  • colours
  • backgrounds
  • borders
  • margins
  • padding
  • box model
  • fonts
  • height/width
  • display

CSS gives you many properties you can add including colour, font style, animation effects, and much more.
